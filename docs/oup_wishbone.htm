3. Register description

3.1. Pin direction register

HW prefix: gpio_ddr
HW address: 0x0
C prefix: DDR
C offset: 0x0

A register defining the direction of the GPIO potr pins.

DDR[7:0]
  • DDR [read/write]: Pin directions
    Each bit in this register defines the direction of corresponding pin of the GPIO port.

    1 means the pin is an OUTPUT

    0 means the pin is an INPUT
